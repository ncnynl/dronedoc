MoveIt!を使ってドローンの経路計画を行う
=================================================================
このチュートリアルでは、ロボット用の経路計画ライブラリである、`MoveIt! <https://moveit.ros.org/>`_ を使ってドローンの経路計画を行う手順について解説します。

**インストール**

以下のコマンドでMoveIt!関連のパッケージをインストールします。

.. code-block:: bash

    sudo apt install ros-kinetic-moveit-*
